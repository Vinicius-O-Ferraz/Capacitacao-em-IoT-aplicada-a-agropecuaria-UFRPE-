# Capacitação em IoT aplicada a agropecuária de precisão (UFRPE)

Este repositório será usado para o desenvolvimento das atividades do curso de Capacitação em IoT aplicada a agropecuária ministrada pelo professor Victor Medeiros na UFRPE.

Esse curso é oferecido pela FACEPE na modalidade PIBITI. A carga horária desse curso é estimado em 30 horas. Divididos em 5 semanas.

Após esse período, haverá seleção de 1o estudantes bolsistas, durante até 3 meses, para o aperfeiçoamento de soluções de IOT nas Startups Semine e Smart Rural.

# Conteúdo Programado

    Semana 1 (6h): Introdução à IoT, plataformas abertas e linguagem C no desenvolvimento IoT ✅
    Semana 2 (6h): Conceitos básicos de eletrônica aplicados ao desenvolvimento de sistemas IoT para agropecuária de precisão
    Semana 3 (6h): Sensores e atuadores comumente utilizados na agropecuária de precisão
    Semana 4 (6h): Redes de comunicação e infraestrutura de nuvem para sistemas IoT aplicados à agropecuária de precisão
    Semana 5 (6h): Prototipação de um projeto telemétrico para a agropecuária de precisão.

  Para cada semana será criada uma branch que conterá um resumo do conteúdo estudado, bem como evidências do projeto desenvolvido.
